CoderDojo とは

CoderDojo は7歳から〜17歳を対象にした非営利のプログラミング道場です。2011年にアイルランドで始まり、2019年7月現在、世界では110カ国・2,000の道場、日本には178以上の道場があります。CoderDojo は有志のボランティアの方々によって各地で運営されていて、無料で参加することができます。CoderDojo では、参加者をニンジャ、教えてくれる人をメンター、各Dojoを運営している人をチャンピオンと呼びます。

CoderDojo あしやについて

兵庫県芦屋市の市民活動センター(リードあしや)様のご協力の下、毎月土曜日に開催しています。(参加する場合は事前に申し込みが必要です)CoderDojo あしやでは、メンターのみなさんが、Scratch や micro:bit、HTML や CSSを使った Web サイト制作などをサポートします。CoderDojo では「子どもたちの作りたいものをサポートする」という目的のもと、ものづくりのサポートを行います。こちらでカリキュラムを組んで、教えているわけではありません。希望に応じて、参考書や教材などを提供することは可能です。

注意事項

  • Wi-Fiに接続可能なパソコンの持ち込みが必要です。
  • 施設での飲食は禁止となっています。水分補給のみ可能です。
  • 事前に Doorkeeper で申し込みをして頂く必要があります。
  • 小学生以下のお子様の場合は、保護者の方の送迎をお願いします。
  • 利用施設に人数制限がございます。参加者多数の場合は、保護者の方々に立ち見、もしくは退席をお願いすることがございます。
  • パソコンにパスワードを設定している場合は、事前にお子様にお伝えいただくか、無効にしていただきますようお願いします。
  • 休憩時間はとっておりません。各自ご自由にお願いします。
  • 貴重品の管理はご自身で行ってください。紛失・破損などの責任は一切とりかねます。
  • 活動の記録のため、写真や動画を撮影いたします。
  • 撮影した写真は広報の一環として Web サイトや SNS などに公開させていただきます。
  • 発表の時間に作ったものを発表することができます。
  • プロジェクターを使用するため、HDMI での接続に対応しているパソコンをご用意いただくか、変換アダプタのご持参をお願いします。

参加するには

こちらより申し込みの方をお願いいたします。 申し込みフォームに自由創作タイムにしたいことを書いてください。 初参加でなにすればいいかわからないという方には、Scratchのチュートリアルのサポートをさせていただきます。

キャンセルをご希望の方

キャンセルの場合は、必ず Doorkeeper よりキャンセルのほうをお願いします。 ひとりでも多くの方にご参加いただきたいと考えておりますので、ご協力よろしくお願いいたします。 キャンセルの方法がわからない場合は下記のメールアドレスよりご連絡いただきますようお願いいたします。

見学について

雰囲気をつかんでいただくために見学も可能です。 ただし参加者の人数によって、お断りさせていただくこともございます。 ご希望の方は下記のメールアドレスよりご連絡ください。

寄付のお願い

CoderDojo あしやでは、活動継続のための寄付を受け付けています。 会場に募金箱を設置しておりますので、任意で募金をお願いします。 ご支援いただいたお金は、会場費や教材費、サイト運営費などの経費に使わさせていただきます。

団体・法人様へ

団体・法人様向けにスポンサー制度をご用意しております。 ホームページや Doorkeeper のイベントページなどにスポンサーリンクを設置させていただきます。 詳しくはホームページ、またはメールでお問い合わせください。

メンターになるには

子どもたちのプログラミングでのものづくりをサポートしてくれるメンターを募集しています。 職業や年齢は問いません！ 教えるのが好き、プログラミングのスキルを活かしてみたいという方は是非ご申請ください。 メンター申請フォーム
